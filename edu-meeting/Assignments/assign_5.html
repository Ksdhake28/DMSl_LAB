AIM:

Design & Develop DB for “Order Management System” with all the constraints.

PROBLEM STATEMENT / DEFINITION:

Design & Develop DB for “Order Management System” with all the constraints. (There must be At least 3 entities and relationships between them.) The statement should use SQL objects such as Table, View, Index, and Sequence.

OBJECTIVE:

  1. Apply DCL and DDL commands to convert ER/EER diagram to tables.
  2. To understand the technique for converting ER diagram into tables
  3. To understand use of DDL , DCL

CONCLUSION:

Understand to design and develop relational database system by using MySql.



Input

  1. CREATE DATABASE
  2. Create Necessary Tables
  3. Insert Data
  4. Create View as required
  5. Give indexing

Output

  1. Table Structure created in database mysql statement
  2. All Database inserted in tables
  3. ER diagram is converted in Tables
